CS BackTop is a lightweight wordpress plugin to integrate a customizable Back to Top feature to any website with ease.

== Description ==

CS Back Top plugin is developed using jQuery Back Top Plugin by Codesalsa. Activating the plugin will integrate a beautiful looking Back to Top button at the right footer position when the page scrolls down. When clicked the button, it gently roll the site to the top. Integrate it to your Wordpress site without any template modifications.

Supported features:

* Smooth scrolling animation, powered by Codesalsa jQuery Back Top plugin
* Customizable options via the admin panel (Scroll Position from Top, Icon Appearance Speed)
* Works on almost all browsers
* No need to add any markup to your theme
* Multiple Icon styling
* Multiple Icon sizes
* Supporting Responsive layouts

== Installation ==

* You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory the top-level folder. 
* Activate the plugin through the 'Plugins' menu in WordPress
* Visit your CS Back Top Settings page
* Configure any Settings as desired, and then enable the plugin
* That's it!
